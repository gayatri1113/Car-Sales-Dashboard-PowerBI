# Car-Sales-Dashboard-PowerBI
A Power BI dashboard project visualizing car sales performance using interactive charts and KPIs. It provides insights into total sales, average price, cars sold, dealer performance, and company-wise sales trends.

Car Sales Dashboard is an interactive data visualization project built in Power BI using car sales data from Excel.
The dashboard helps analyze key business metrics like:
Year-to-Date (YTD) and Month-to-Date (MTD) sales
Average car prices and cars sold
Sales performance by company, dealer, color, and body style
Regional insights with a map view
Weekly sales trends across the year
The goal of this project is to transform raw car sales data into actionable insights through a visually engaging and easy-to-navigate Power BI dashboard.
